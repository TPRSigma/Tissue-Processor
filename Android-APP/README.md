## ðŸ“² Installing an APK on Android (build)

This guide explains how to install an Android application directly from an `.apk` file, without using the Google Play Store.

---

### 1. Enable Installation from Unknown Sources

1. Open **Settings** on your Android device.
2. Go to **Apps** or **Apps & notifications**.
3. Find **Special app access** or **Install unknown apps**.
4. Select the app you will use to install the APK (e.g., **Browser** or **File Manager**).
5. Enable **Allow from this source**.

> âš ï¸ Warning: Installing apps from unknown sources can be a security risk. Only install trusted APKs.

---

### 2. Transfer the APK to Your Device

- Move the `.apk` file to your device via USB, email, cloud storage, or Bluetooth.
- Recommended location: **Downloads** folder or a folder you commonly use for files.

---

### 3. Open the APK File

1. Open a **File Manager** or **Downloads** app on your device.
2. Locate the `.apk` file you want to install.
3. Tap the file and confirm the installation.

---

### 4. Complete Installation

- Follow the installer prompts.
- Once finished, you can open the app directly or find it on your home screen.

---

### 5. Tips

- If the installation fails, make sure the `.apk` file is not corrupted and is compatible with your Android version.
- After installation, consider disabling **installation from unknown sources** for security.
